# Лабораторная работа №6

**Тема:**
Программирование с использованием функций.

**Задача:**
Вывести на экран таблицу значений функции и ее разложения в ряд s(x)
для x, изменяющегося от a до b с шагом h = (b – a) / 10. Расчет y(x)
и s(x) поместить в функцию. Использовать прототипы функций. Параметры
передавать указанным в соответствии с вариантом (по ссылке).
Расчет функции s(x) выполнить с заданной точностью eps.
